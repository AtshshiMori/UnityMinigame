●サンプルデータについて
サンプルデータには『UNITY5 3Dゲーム開発講座　ユニティちゃんで作る本格アクションゲーム』の本文中で解説したサンプルが収録されています。

本書では、2015年1月時点でUnity5を予約した方向けに提供されているUnity5 β19で、CHAPTER 02、CHAPTER 04、CHAPTER 05、CHAPTER 06、CHAPTER 07、CHAPTER 08のサンプルを作成、検証をしています。
なお、APPENDIX 01では、Live 2Dを利用して動くユニティちゃんを作成手法を解説しています。APPENDIX 02では、Unreal Engineでユニティちゃんを利用する方法について解説しています。

●サンプルのディレクトリ構成について
P.008にあるように「UNITY5」フォルダにまとめてアップロードする予定でしたが、各章の容量が大きいので章単位で分けてアップロードしています。

●本書の解説で利用しているUnityのバージョンについて
本書は2015年1月時点でUnity5を予約した方向けに提供されているUnity5 β19でサンプルを作成、検証をしています。今後リリースされるUnity5の正式版とは画面構成が異なる場合がありますので、あらかじめご了承ください。

●サンプルの検証環境
2015年1月現在、Unity 5.0 β19の開発環境は以下のとおりです。

・Windows版
　Windows XP SP2 以降の Windows

・Mac版
　Mac OS X Intel CPU および 10.6 以降の Mac OS

なお、本書ではWindows版にてサンプルの作成／動作検証および誌面の解説しています。Mac版の場合、誌面と操作手順が異なる場合があります。またMac版におけるサンプルの動作検証はしていません。あらかじめご了承ください。
